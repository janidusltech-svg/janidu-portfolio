# Janidu Sandaruwan Portfolio Website

A dark modern personal portfolio website for Janidu Sandaruwan.

## Run locally

Open `index.html` directly in your browser, or use VS Code Live Server.

## Recommended Git workflow

Before work:
```bash
git pull
```

After work:
```bash
git add .
git commit -m "Update portfolio"
git push
```

## Publish options

### Vercel
1. Push this folder to a GitHub repository.
2. Go to Vercel and import the GitHub repository.
3. Deploy.
4. Add your custom domain in Vercel project settings.

### GitHub Pages
1. Push this folder to a public GitHub repository.
2. Go to repository Settings > Pages.
3. Select the branch and root folder.
4. Save and publish.
