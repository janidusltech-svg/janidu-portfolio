# Janidu Sandaruwan Portfolio + Study Hub By J

This project contains Janidu's portfolio website and the Study Hub By J student-resource page.

## Public Study Hub behavior
- Public visitors can only view/open/download resources.
- There is no public upload form.
- Currently visible active section: Year 1 Semester 2.
- Year 1 Semester 1, Year 2, Year 3, and Year 4 are marked as Coming Soon.

## How Janidu adds resources privately
1. Put HTML quizzes in `quizzes/`.
2. Put PDFs/reference sheets in `downloads/`.
3. Add the resource details to `data/resources.json`.
4. Commit and push to GitHub. Vercel will redeploy automatically.
